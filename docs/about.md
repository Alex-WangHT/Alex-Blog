# 👨‍🔬 关于我和我的后院瓜田
你好，欢迎来到我的后院瓜田 🍉！

我相信，所有伟大的工程实践都源于扎实的理论记录和不懈的尝试。这个“瓜田”就是我用来记录和沉淀知识的地方。

## 😎 我是谁？ (Who am I?)
你好，你可以叫我**Alex.Wang**~~~（以后会补充自己的个人简历啦）~~~

我是一名默默无闻的在读博士，同时也是一个预备役的机器人工程师。
## 🛰️ 研究聚焦 (Research Focus)
我的研究方向是：**自主空中机械臂 (Autonomous Aerial Manipulators)**。

这是一种**多旋翼无人机平台下面带机械臂的~~~奇形怪状~~~的机器人**。它结合了多旋翼无人机的机动性与机械臂的精细操作能力，旨在实现复杂环境下的高空作业。

## 🚀 我的目标 (The Mission)
我最近的核心目标是：

!!! success "将空中机器人从室内实验的枯燥无聊的环境中解救出来，让它们真正在实际场景中落地。"
## 📝 为什么记录 (Why This Blog)
这个博客是我的实践档案，旨在完整记录**自主空中机械臂从理论走向实际落地**的全部过程。它涵盖了以下核心要素：

- 📘 技术路线图 (Roadmap): 记录实现这一宏伟目标背后的技术选型、架构设计、路线规划和关键工程决策的演进轨迹。

- 💡 经验积累 (Insights): 深入分享在运动控制、自主导航和具身智能等前沿领域中提炼出的工程实战经验、避坑指南和算法优化的独到心得。

- 🌱 持续成长 (Growth Log): 不仅是成功的证明，更是失败的记录和挑战的解析，体现一个工程师不断学习、自我迭代与突破的真实历程。

我希望通过日复一日的耕耘，让这片后院瓜田最终能硕果累累，不仅为我自己提供查阅，也能为所有对空中机器人领域感兴趣的同仁提供参考和启发。
